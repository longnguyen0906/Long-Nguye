CƠ BẢN VỀ LIST

ORDER LIST

Sử dụng thẻ OL và LI để định nghĩa danh sách

  1. item 1
  2. item 2
  3. item...

Sử dụng attribute START để set số bắt đầu

  1. item 3
  2. item 4
  3. item...

Sử dụng attribute VALUE trong thẻ LI để reset lai giá trị bắt đầu

  1. item 7
  2. item 8
  3. item 3
  4. item 4

Sử dụng attribute REVERSED để đảo chiều đếm

  1. item đầu
  2. item n-1
  3. item....
  4. item cuối

Sử dụng attribute TYPE để thay đổi kiểu
Các giá trị cho kiểu gồm : "1|a|A|i|I"

  1. item đầu
  2. ....
  3. item cuối

Nested List

  1. item 1
  2. item 2
  3. item 3
    1. item C1
      1. item 11
      2. item 12
      3. item 13
    2. item C2
    3. item C3
  4. item 4

UNORDER LIST

Sử dụng UL và LI để định nghĩa danh sách

  • item 1
  • item 2
  • item 3

Nested List

  • item 1
  • item 2
  • item 3
    • item 3.1
    • item 3.2
    • item 3.3
  • item 4

DESCRIPTION LIST

Sử dụng DL, DT và DD để định nghĩa danh sách

item 1
value 1
item 2.1
item 2.2
value 2
item 3
value 3.1
value 3.2
