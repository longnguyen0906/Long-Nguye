ATTRIBUTE OF IMAGE MAP

THẺ IMG

src: đường dẫn đến hình ảnh
usemap: sử dụng tên thuộc tính name ở thẻ map (thêm # như ID)

THẺ MAP

name: như ID
THẺ AREA (thẻ con map): định nghĩa các vùng
shape: hình dạng các vùng
coords: tọa độ các vùng (mở hình gốc để xác định tọa độ)
href: đường link liên kết
